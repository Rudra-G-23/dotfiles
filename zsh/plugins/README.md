# Zsh Plugins Overview

Here is a quick reference table explaining why and when to use the plugins configured in this toolkit.

| Plugin | When to use it | Why use it |
|---|---|---|
| **Starship** | Always (Prompt) | Provides a customizable, fast, cross-shell prompt with rich context (Git branch, Python version, etc.). |
| **zsh-autosuggestions** | Typing commands | Suggests commands based on your terminal history. Saves typing and helps recall previous commands. |
| **zsh-syntax-highlighting** | Typing commands | Highlights commands in real-time. Immediately catches typos before pressing Enter. |
| **fzf** | Searching | Fuzzy finding files, command history, processes, etc. Press `Ctrl+R` to find anything extremely fast. |
| **zoxide** | Navigating directories | A smarter `cd` command. Remembers frequently used directories so you can jump to them instantly. |
| **eza** | Listing files (`ls`) | Modern replacement for `ls` with colors, icons, git status, and tree views. |
| **bat** | Viewing files (`cat`) | Modern replacement for `cat` with syntax highlighting and line numbers. |
